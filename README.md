## 📂 Structure

Solutions are grouped into folders by **difficulty range**:

```
Eolymp/
├── 0-1000/
├── 1000-2000/
├── 2000-3000/
├── ...
└── 10000+/
```

This makes it easy to browse problems from beginner level all the way up to the hardest tiers.

## 🎯 About

A comprehensive practice archive from the [Eolymp](https://www.eolymp.com/) judge, spanning a wide spread of topics and difficulty levels — from introductory exercises to advanced algorithmic challenges. Solutions aim to be efficient and readable.

## 🛠️ Language

Primarily **C++** (with occasional Python).

## ▶️ Compile & Run

```bash
g++ -O2 -std=c++17 solution.cpp -o sol
./sol
```
